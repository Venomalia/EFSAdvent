CIRC (Magic circle)
Appearing magic circle warp

Variable 1a:
The local variable number that triggers or activates this actor.

Variable 1b:
0 = Back To TV
1 = Slight shaking, Pixelation transition and music distortion.
2 = Heavy shaking
3 = Slight shaking
4 = no shaking

Variable 2:
The X coordinate players arrive at

Variable 3:
The Y coordinate players arrive at

Variable 4:
0 = Default
1 = Green
2 = Red
3 = Blue
4 = Purple
5 = 10 seconds
7 = Cactus
8 = Glitter effect
16 = Disappears after all players have used it.
