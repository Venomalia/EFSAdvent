SWOR (OR switch check)
Sets the local variable specified in V1 if any of the local variables specified in V2 or V3 are set.

Variable 1:
The local variable number that is triggered or set by this actor.

Variable 2:
A local variable number that is checked by this actor.

Variable 3:
A local variable number that is checked by this actor.
