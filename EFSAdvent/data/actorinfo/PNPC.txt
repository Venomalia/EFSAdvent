PNPC (Panel rewriting NPC)
Causes a tile to be changed to a different tile.

Variable 1a:
Variable that activates the tile to be changed.

Variable 1b:
0 = No special effect
4 = Dust particles

Variable 2:
0 = Base layer
1 = Top layer
16 = ?

Variable 3+4:
Value of the tile that appears.

Packed data is composed as follows:
